Apple Spyder

Simple web entry for Apple Spyder. The routes below are useful for manual checks, feed preview, and notification testing.

  • /apple-spyder/software-release — Check Apple Developer RSS feed and send notifications if new items are found.
  • /apple-spyder/software-release/feed — Fetch current Apple Developer RSS entries.
  • /apple-spyder/accessory-ota-update — Check Apple accessory OTA updates.
  • /apple-spyder/test-notify — Send a test notification with the default message.
